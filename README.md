# Playwright QA Automation Portfolio 🚀

This project is a high-level automated testing suite for **SauceDemo**, focusing on scalability, maintainability, and clean code.

### 🛠️ Tech Stack
* **Framework:** Playwright
* **Language:** TypeScript
* **Pattern:** Page Object Model (POM)
* **Principles:** DRY (Don't Repeat Yourself), Data-Driven Testing.

### 🏗️ Current Progress (In Construction)
- [x] Base Architecture (POM)
- [x] Login Feature (Happy path & Error handling)
- [x] Inventory Feature (Product validation & Cart logic)
- [ ] Checkout Workflow (E2E) - *Next step*
- [ ] GitHub Actions Integration (CI/CD)

### 🚀 How to run
1. `npm install`
2. `npx playwright test`
